Equivalencias de Datos entre Brasil y España para PolicySpace2

Adaptación completa de fuentes de datos para el contexto español

Este sitio web proporciona una guía completa para adaptar el proyecto PolicySpace2 al contexto español. Hemos identificado todas las fuentes de datos españolas equivalentes a las utilizadas en el proyecto original brasileño, incluyendo enlaces directos a las APIs y recursos necesarios.

La información está organizada por categorías para facilitar la adaptación del proyecto, y se proporciona código Python para conectar con las APIs y fuentes de datos españolas.

Resumen de Equivalencias

Se han identificado equivalencias para los 138 documentos originales del proyecto PolicySpace2, agrupados en las siguientes categorías:

  • Datos Geográficos y Administrativos: Códigos territoriales, relaciones entre municipios y provincias
  • Datos Demográficos: Población, fertilidad, mortalidad y matrimonio
  • Datos Económicos: Empresas, financiación municipal e indicadores económicos
  • Datos Educativos: Nivel educativo por municipio

Código de Ejemplo para Conexión con API del INE

El siguiente fragmento de código muestra cómo conectar con la API JSON del INE para obtener datos demográficos:

import requests
import pandas as pd

def obtener_datos_ine(funcion, input_id, idioma="ES", parametros=None):
    """
    Obtiene datos de la API JSON del INE
    
    Args:
        funcion (str): Función a utilizar (DATOS_TABLA, DATOS_SERIE, etc.)
        input_id (str): ID de entrada para la función
        idioma (str): Idioma (ES o EN)
        parametros (dict): Parámetros adicionales
        
    Returns:
        dict: Datos en formato JSON
    """
    base_url = f"https://servicios.ine.es/wstempus/js/{idioma}/{funcion}/{input_id}"
    
    if parametros:
        params_str = "&".join([f"{k}={v}" for k, v in parametros.items()])
        base_url += f"?{params_str}"
    
    response = requests.get(base_url)
    if response.status_code == 200:
        return response.json()
    else:
        print(f"Error {response.status_code}: {response.text}")
        return None

# Ejemplo: Obtener datos de población
datos_poblacion = obtener_datos_ine("DATOS_TABLA", "t1", parametros={"year": "2023"})

# Convertir a DataFrame
if datos_poblacion and "Data" in datos_poblacion:
    df = pd.DataFrame(datos_poblacion["Data"])
    print(df.head())
    
    # Guardar a CSV
    df.to_csv("poblacion_municipios_2023.csv", index=False)
